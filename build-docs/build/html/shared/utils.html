3. Utils helpers¶

Utils contains all the common helpers that every service can use. It consist of some general functions like getting a dict obj from a .json file.

utils.get_current_thread_name()¶

Get name of current running threads.

Returns

Thread name

Return type

str

utils.get_json(file)¶

Retrieve the content of a json file

Parameters

file (str) – absolute root from project path

Returns

The dictionary with content of the file

Return type

dict

retrieve absolute path from root directory

The absolute path

utils.safe_run(func, args=[], kwargs={})¶

Returns _SafeRunError() upon failure and logs stack trace.

Parameters

  • func (callable) – The function that needs to be called safely

  • args (list, optional) – The args. Defaults to [].

  • kwargs (dict, optional) – The kwargs. Defaults to {}.

Returns

return exception if it fails and returns function result if it succeeds

Return type

Exception|any
