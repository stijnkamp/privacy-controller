5. Websocket Service¶

5.1. Module contents¶

class websocket.Websocket(state)¶

Bases: thread.Thread

Service to start a websocket which can send updated to the front-end in real-time. Clients can register to specific channels and it will get updates for those messages through a websocket.

async check_for_stop()¶

Keep running until thread is not active

async handler(websocket, path)¶

Handle a connection and dispatch it according to who is connecting.

Parameters

  • websocket (WebsocketClient) – The websocket client that just connected.

  • path (str) – The path to which the client subscribed

async notify_clients()¶

Watch the queue and send message from queue to registered clients
